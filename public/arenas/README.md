- `citadel.jpg`   — Šviesos pulkas (mithrilo tvirtovė)
- `inferno.jpg`   — Demonų orda (pragaras / lava)
- `scrapyard.jpg` — Goblinų gauja (laužynas / urvas)
- `harbor.jpg`    — Plėšikų naktis (naktinis uostas)
