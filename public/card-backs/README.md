# Kortų nugarėlės (miniatiūros)

Įkelk čia šiuos failus (kodo keisti nereikia – file-first, fallback į dabartinę tamsią plokštelę):

- `back.webp`  — paprasta kortos nugarėlė (kaladė, priešo ranka)
- `curse.webp` — prakeiksmo nugarėlė (reakcijų/spąstų slotai)
- `zmk.webp`   — ŽMK nugarėlė (ŽMK krūva)

Rekomendacija: portretinė ~3:4 proporcija (pvz. 300×400+), WEBP/PNG. Naudojama ir desktop, ir mobile.
Kapinynas rodo paskutinės į jį patekusios kortos paveikslėlį automatiškai (nereikia nugarėlės).
